# Chunk 類型速查

切分句子時，先掃一遍有沒有以下類型。它們都是「多個字當一塊讀」，是使用者最常切錯的地方。

## 1. 量詞 / 部分框架（partitive frames）

`a … of …` 結構，整塊是「一組數量」，中間的名詞往往不是字面意思。

| 框架 | 意思 | 注意 |
|---|---|---|
| a set of | 一組、一批 | set ≠「設定」 |
| a range of | 一系列、各種 | range ≠「範圍」字面 |
| a body of | 大量（研究、水、人） | body ≠「身體」 |
| a wealth of | 大量、豐富的 | wealth ≠「財富」 |
| a piece of | 一則、一項 | a piece of advice/news |
| a couple of | 幾個、兩三個 | 不是剛好「一對」 |
| a handful of | 少數幾個 | — |

可加形容詞：`a diverse set of` / `a wide range of` / `a growing body of`，形容詞修飾的是「這一批」。

## 2. 片語動詞（phrasal verbs）

動詞 + 介系詞/副詞，意思常常不等於字面相加。

| 片語 | 意思 |
|---|---|
| come from | 來自、出身 |
| come up with | 想出、提出 |
| carry out | 執行 |
| account for | 占（比例）、解釋 |
| break down | 故障、分解、拆解 |
| make up | 組成、編造、化妝 |

⚠️ 警訊：動詞後面緊跟 `up/out/on/off/in/down/for/with` 時，先假設它們是一塊。

## 3. 固定搭配（collocations）

某些字習慣一起出現，硬拆會怪。

`make a decision`（做決定，不用 do）、`heavy rain`（大雨，不用 big）、`take into account`（納入考量）、`pay attention to`（注意）。

## 4. 複合名詞（compound nouns）

兩個以上名詞組成一個概念，**重點在最後一個字**，前面的是修飾。

`customer service`（客服）、`climate change`（氣候變遷）、`supply chain`（供應鏈）、`machine learning model`（機器學習模型 ← 三個字一個概念）。

讀法：從後往前 →「⋯⋯的⋯⋯的 X」。

## 5. 慣用語 / 固定表達（idioms & fixed expressions）

整體意思和字面無關，**絕對不能逐字翻**。

`in terms of`（就⋯⋯而言）、`as well as`（以及）、`on the other hand`（另一方面）、`when it comes to`（談到、說到）、`a wide variety of`（各式各樣的）。

## 常見「切錯」陷阱

使用者把 `a b c` 誤切成 `a b` + `c`，通常因為沒認出 `b c` 是一塊。檢查順序：

1. **`of` 出現** → 往前找，可能是量詞框架 `a … of`。
2. **動詞後接小介系詞** → 可能是片語動詞。
3. **連續名詞** → 可能是複合名詞，當一塊讀。
4. **某個字「字面意思套不進去」** → 它多半是某個固定表達的一部分，不是獨立的字。

切分原則：**寧可先當成一塊，解釋時再拆**——而不是先逐字拆，因為逐字拆正是看不懂的原因。
